# Taskfile Commands

## Main commands

```bash
task install
task dev
task check
task ci
```

## Frontend

```bash
task fe:lint
task fe:typecheck
task fe:test
task fe:build
```

## Rust

```bash
task rs:fmt
task rs:clippy
task rs:test
task rs:check
```

## Quality

```bash
task q:file-size
task q:color-tokens
```
